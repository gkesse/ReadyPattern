# ReadyPattern

**ReadyPattern** est une bibliothèque de fonctions C++ pour le développement de Patrons de Conception.  
C'est une bibliothèque développée en C++ et maintenu par 
[Gerard KESSE](http://5.48.215.193:8855/presentation/ "Accédez à mon site web (ReadyDev)").

# Technologies

Le développement de la bibliothèque **ReadyPattern** a nécessité les technologies suivantes :

* Langage : **C++**
* IDE : **Visual Studio**
* Outil de Construction : **NMake, Cygwin**
* Editeur de Texte : **Notepad++**
* Concept DSP : **Singleton, Fabrique, Observateur, Stratégie**
* Cible : **Windows, UNIX**
